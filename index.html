Oryol Emscripten Samples

Oryol on Github

Please note that these samples may be using bleeding edge features, thus they may not work even in the latest public/stable browser version but may require the bleeding edge versions like Firefox Nightly or Chrome Canary!

These should run in any WebGL-enabled web browser:

  • Clear: Just call glClear in a loop
  • Triangle: The canonical triangle demo
  • Shapes: Shapes with random vertex colors, created with ShapeBuilder class
  • PackedNormals: Shapes with packed vertex normals
  • SimpleRenderTarget: Demonstrate simple offscreen rendering
  • InfiniteSpheres: Infinite recursive spheres
  • DDSTextureLoading: Load and render various DDS texture formats
  • DDSCubeMap: Load and display DDS DXT1 cube map
  • DrawCallPerf: Measure draw call performance
  • FullscreenQuad: Fullscreen quad with some distance field ray-marching
  • PBRendering: Physically based rendering sample
  • Instancing: Instanced rendering
  • TextureFloat: RGB plasma using float render texture
  • VertexTexture: Vertex texture fetch from offscreen render target
  • GPUParticles: Fully GPU driven 3D particles
  • HTTPClientSample: Low-level HTTP loading sample
  • CoreHello: Low-level hello world sample
  • DebugText: Demonstrate debug text rendering
  • TestInput: Test Input module functionality
